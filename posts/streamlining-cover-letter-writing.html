Streamlining Cover Letter Writing

July 6, 2020 - 6 minute read -
Python LaTeX Career

In the job search process, one inevitably has to produce application materials, including a custom cover letter, and keep them organized. We can use a Python along with a cover letter template (typset in LaTeX) to automatically produce folders and the necessary documents.

Our goal will be to construct a directory with the following structure. The script addJob.py creates a new folder with the TeX file and compiles it with pdflatex, producing the PDF output.

Using a LaTeX cover letter template

To auto-generate documents, I used a combination of Python and the following LaTeX cover letter template based on one created by Gregor Taube stored as coverLetterTemplate.tex. While there are more sophisticated packages available, the most simple method is using Template module from the string package. We take user inputs from the command line. Templates support “$”-based substitutions, but since “$” is reserved in LaTeX for typesetting math, we’ll use “»”. This way we can tailor the cover letter to each job by substituting over the tags.

\documentclass[11pt,a4paper,roman]{moderncv}
\usepackage[english]{babel}

\moderncvstyle{classic}
\moderncvcolor{green}

% character encoding
\usepackage[utf8]{inputenc}

% adjust the page margins
\usepackage[scale=0.75]{geometry}

% personal data
\name{Orton}{Babb}

\phone[mobile]{(My Number Here)}
\email{ortonbabb@gmail.com}

\begin{document}

\recipient{To}{>>employer}
\date{\today}
\opening{Hi!}
\closing{Thanks in advance!}
\enclosure[Attachment]{Résumé}
\makelettertitle


% The first paragraph of your job application letter should include information on why you are writing. Mention the job you are applying for and where you found the position. If you have a contact at the company, mention the person's name and your connection here.

I want to introduce myself as a candidate for the >>position position. The firms’s mission to >>mission resonates with me. I am a recent graduate with a major in mathematics, minor in computer science and a motivation to >>role. Given this background, I am excited to apply to the >>positionalt position to >>jobdescription.

% The next section of your cover letter should describe what you have to offer the company. Make strong connections between your abilities and the requirements listed in the job posting. Mention specifically how your skills and experience match the job. Expand on the information in your resume, don't just repeat it.

% Try to support each statement you make with a piece of evidence. Use several shorter paragraphs or bullets rather than one large block of text, which can be difficult to read and absorb quickly.

You look for a person with >>abilities, I am certainly such a person, and can support the claim with several examples
\begin{itemize}
>>justifications
\end{itemize}

State that you will do so and indicate when (one week's time is typical). You may want to reduce the time between sending out your resume and follow up if you fax or email it.

I look forward to hearing from you. Please do not hesitate contacting me if you have any questions.

\makeletterclosing

\end{document}

Formatting the cover letter template

Next we’ll need to create a Python script, call it addJob.py, to make the substitutions from the terminal. To keep it simple, we use the os package to also take care of creating the folder to store the results and make a system call to pdflatex.

from string import Template
import os

# Create a new class with the ">>" delimiter to avoid conflicts with LaTeX
class LaTeXTemplate(Template):
    delimiter = '>>'

# Read in the template
with open('coverLetterTemplate.tex', 'r', encoding='utf') as file:
    coverLetterTemplate = LaTeXTemplate(file.read())

# Get the inputs to make substitutions in the template

#   first paragraph
print('Employer: '); employer = input(); print('\n')
print('Position: '); position = input(); print('\n')
print("Mission e.g. 'help alleviate poverty across the globe':"); mission = input(); print('\n')
print("Role e.g. 'study consumer spending habits': "); role = input(); print('\n')
print('Alternate position description:'); positionalt = input(); print('\n')
print("Job description? Give more details, e.g. 'support senior analysts with data collection, pre-procession and analysis': "); jobdescription = input(); print('\n')

#   second paragraph
print("Abilities e.g. 'the ability to do x, y, z'"); abilities = input(); print("\n")
justifications = ""
#       loop until enough justification are added
while True:
    print("Enter justification:"); justification = input();
    if justification == "quit":
        break
    else:
        justifications+= "\item " + justification +  "\n"
        print("\n")

# Apply the substitutions
coverLetter = coverLetterTemplate.substitute( employer = employer, position = position, mission = mission, role = role, positionalt = positionalt, jobdescription = jobdescription, abilities = abilities, justifications = justifications)

# Create a directory and store the results
fname = f"{employer} - {position}"

os.mkdir(fname); os.chdir(fname)

with open(f"{fname}.tex", "w", encoding="utf") as file:
  file.write(coverLetter)

# Compile the LaTeX code
os.system(f"pdflatex '{fname}.tex'")
